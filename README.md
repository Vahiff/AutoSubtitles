# AutoSubtitles

AutoSubtitles is a product repository for an automatic subtitle generation tool. The repository is currently prepared as a clean project space for planning, implementation, testing, and future release packaging.

## Project Status

Status: planning / initial repository

Planned direction:

- import or record audio/video
- generate subtitles automatically
- edit subtitle timing and text
- export SRT, VTT, and burned-in captions
- package for desktop, web, and mobile where useful

## Target Users

- content creators
- small businesses
- video editors
- course creators
- social media teams

## Future Tech Options

- web MVP with React + Vite
- backend processing with Python or Node.js
- speech-to-text integration
- desktop packaging with Tauri or Electron
- mobile packaging with Capacitor or React Native

## Repository Standards

This repository is prepared with professional documentation, security notes, and contribution rules so it can grow cleanly when implementation begins.
